Come funziona l'Auto-Sync in Background

A differenza di altri strumenti, WolfMind non richiede l'installazione di client Git esterni o operazioni manuali. L'applicazione sfrutta un motore scritto in Rust che:

  • Rileva istantaneamente la creazione, modifica o eliminazione di note e sessioni.
  • Gestisce la coda dei salvataggi in background in modo non bloccante.
  • Effettua il git add, git commit automatico con messaggi descrittivi ed esegue il git push silenziosamente.
src-tauri/src/lib.rs 
fn auto_git_commit_and_push(repo_path: &Path, message: &str) -> Result<(), String> {
    // Esegue git add, commit e push tramite std::process::Command
    Command::new("git").args(["add", "."]).current_dir(repo_path)...
    Command::new("git").args(["commit", "-m", message]).current_dir(repo_path)...
    Command::new("git").args(["push"]).current_dir(repo_path)...
    Ok(())
}

Come ottenere Chiavi API Gratuite

Configura il tuo assistente IA in meno di 5 minuti senza spendere nulla.

Groq offre il motore di inferenza più veloce del pianeta con modelli open-source (come Llama 3 e Mixtral) accessibili gratuitamente:

  1. Visita la Groq Console.
  2. Registrati o accedi con il tuo account.
  3. Vai nella sezione "API Keys".
  4. Clicca su "Create API Key", assegna un nome ed effettua la copia.
  5. Incolla la chiave nelle impostazioni di WolfMind.

OpenRouter è un aggregatore che permette di testare moltissimi modelli avanzati:

  1. Registrati su OpenRouter.ai.
  2. Vai nel tuo profilo e seleziona "Keys".
  3. Genera una chiave.
  4. Cerca i modelli contrassegnati con la dicitura "Free" (ad es. Llama-3-8B-Instruct-Free, Gemma 2) per usarli gratis.

Se desideri la massima privacy senza trasmettere dati a servizi di terze parti:

  • Ollama: Scarica Ollama, esegui ollama run llama3 e configura WolfMind all'indirizzo predefinito http://localhost:11434/v1.
  • LM Studio: Avvia il Server Locale all'interno dell'app e connetti WolfMind all'indirizzo http://localhost:1234/v1.
